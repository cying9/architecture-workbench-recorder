# Roadmap

## v0.1.x

- stabilize the pre-plan checkpoint workflow
- keep the default artifact budget small
- improve apply safety and conflict reporting
- make the CLI and skill entrypoints easier to use repeatedly

## Near-Term

- add a first-class `prepare-plan` command
- refine structured transcript normalization for long threads
- improve conflict classification for reopened questions and glossary drift
- add more examples from real product and architecture discussions

## Later

- provider-agnostic extractor integrations beyond the current external command path
- richer planning-focused metadata in the claim ledger
- optional repository-local templates for different workbench styles

## Explicit Non-Goals

- generic meeting minutes
- passive background capture
- autonomous architecture decisions
- heavyweight workflow orchestration for its own sake
